We have the following abstract class:

abstract class Shape {
    abstract fun calculateArea(): Double
    abstract fun calculatePerimeter(): Double

    fun printDescription() {
        println("This is a shape.")
    }
}

Define a concrete class called Square that extends Shape, accepts one parameter as a constructor called side, and performs all operations needed according to the names of its members.


Sample Input:
5

Sample Output:
Area = 25.0
Perimeter = 20.0
